플레이 타임 단축 & 편의성 개선

플레이 타임 단축

  • 보스 제한시간 2/3로 축소 + 보스 최대체력 2/3로 축소 (Ex. 검은마법사 20분 / 어센트 스킬 데미지 축소)
  • 헥사 매트릭스 초기화 주문서 전체 지급
  • 결정석 판매가 5~10% 조정 (결정석 가격 감소)
  • 몬스터파크 소탕모드 — 1일 1회 돌면 다회 돈 만큼의 경험치 지급
  • 익몬 → 익스트림 몬스터파커로 변경 — 일일 몬스터파크 주 5회 플레이 시 익몬 경험치 지급
  • 심볼퀘 단축 — 전부 100마리로 통일
  • 아케인 스페셜 컨텐츠 플레이 주 1회로 단축
  • 에픽던전 5라운드로 압축 — 몬스터 HP 감소 / 주화 보상을 솔 에르다 조각 15개로 변경 / 퀵 패스 가격 50% 감소

  • 길드 시스템 개편 — 월드 단위 가입 가능, 편의성·UI 개편, 6마리까지 같은 길드 가입으로 판정
  • 스킬 UI 개편 — 스킬 포인트 자동 배분 기능
